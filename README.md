# U2A3.-Programa-Perceptron-Multicapa-para-Seguidor-de-Linea
Programa realizado en un archivo Notebook de Python, donde se emplea un perceptrón multicapa (red neuronal pequeña) para el control de un seguidor de linea simulado.

Se emplea Visual Studio code para la creación e implementacion del archivo ipynb, el cual realiza el entrenamiento correspondiente mediante el perceptron multicapa. El resultado son los valores de pesos y bias, los cuales se generan en un archivo txt. Estos datos se ingresan al archivo de codigo Arduino, el cual realiza el control del seguidor de linea simulado en Proteus (TURTLE).
